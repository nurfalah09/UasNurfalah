Sistem CRUD Pemesanan Tiket Bioskop merupakan sebuah aplikasi berbasis Python yang digunakan untuk mengelola proses pemesanan tiket bioskop secara sederhana. 
Sistem ini dirancang untuk memudahkan pengguna dalam melakukan pengelolaan data film dan pemesanan tiket melalui fitur Create, Read, Update, dan Delete (CRUD) yang terhubung dengan database MySQL.

Aplikasi ini menggunakan konsep Object Oriented Programming (OOP) dengan penerapan class dan object, 
serta memanfaatkan modul dan package untuk memisahkan struktur program agar lebih terorganisir dan mudah dipahami. Selain itu,
sistem ini juga menerapkan percabangan (ifâ€“elif) dan perulangan (while) sebagai dasar logika pemrograman dalam pengolahan menu.

Database MySQL dikelola menggunakan phpMyAdmin, sehingga data film dan pemesanan tiket dapat disimpan, ditampilkan, dan dikelola secara terstruktur.
Sistem ini berjalan berbasis console (terminal) dan ditujukan sebagai media pembelajaran maupun tugas akademik
untuk memahami integrasi Python dengan database MySQL.<img width="1052" height="747" alt="image" src="https://github.com/user-attachments/assets/a8cb71db-e63c-4ca8-bfc0-a5558f49386f" />

Tujuan dari pembuatan sistem ini adalah:

Memahami konsep CRUD menggunakan Python dan MySQL

Menerapkan Object Oriented Programming (OOP) dalam studi kasus nyata

Melatih penggunaan modul dan package dalam pemrograman Python

Mengintegrasikan Python dengan database MySQL melalui phpMyAdmin
